ReVanced

YouTube ReVanced (or simply ReVanced) is a modified third-party from discontinued project of YouTube Vanced application for Android with a built-in ad blocker. Other features of the app include SponsorBlock, background play, free picture-in-picture (PiP), an AMOLED black theme, swipe control for brightness and volume, and the ability to restore dislike counts on YouTube videos. A YouTube Music version of ReVanced was also developed.

















Overview

YouTube ReVanced is a modded version that change discontinued project of Vanced. It adds a number of features that you will never find in the official YouTube app from Google. Things like built-in adblocking, background playback without YouTube Premium, black/dark themes, and much more.

You can also visit ReVanced GITHUB or Discord for more information about This Project. Also thanks to XDream8 who already provide me the package to compile it with ReVanced :D
